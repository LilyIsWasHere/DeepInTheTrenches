Controls:

=== SPAWNING UNITS ===

ALT + LMB: Spawns player units
ALT + RMB: Spawns enemy units

Clicking these numbers modifies the type of player/enemy unit you will spawn next time you use the ALT + MB control scheme:

1: Switches to foot unit
2: Switches to turret unit
3: Switches to mortar unit
4: Switches to production unit

=== CONTROLLING UNITS ===

RMB: When clicked on a unit, it will show the available actions and allow you to set it (for example: Move will ask you to select a target position and will then move to that position).
LMB: When dragged, it will select all the units within the area selected and allow you to control all at once rather than each individually.

=== EXCAVATION PATH ===

LMB: When clicked or dragged, creates excavation path (not yet available as an action, but there to be shown, it will also show up when dragging to select units but it won't interfere with anything)
CTRL: Resets/removes excavation path

=== MODIFY TERRAIN ===

SHIFT + LMB = Grow terrain (add terrain to clicked position)
SHIFT + RMB = Dig terrain (remove terrain to clicked position and collect resources accordingly, resources dug up can be viewed in the console)
